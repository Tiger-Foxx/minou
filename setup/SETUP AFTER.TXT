IMMEDIATE NEXT STEPS:
┌─────────────────────────────────────────────────────────────────────────┐
│  1. 🚀 LAUNCH MINOU                                                    │
│     • Double-click the desktop shortcut "Minou"                        │
│     • Or go to Start Menu → Programs → Minou → Minou                   │
│                                                                         │
│  2. 🐾 MEET YOUR PET                                                   │
│     • Your pet will appear somewhere on your screen                    │
│     • Look for the animated cat or dog character                       │
│     • Check the system tray for the Minou icon                         │
│                                                                         │
│  3. 💬 START CHATTING                                                  │
│     • Double-click on your pet to open the chat                        │
│     • Or double-click the system tray icon                             │
│     • Try saying "Hello!" or "What's your name?"                       │
└─────────────────────────────────────────────────────────────────────────┘

QUICK CONTROLS:
┌─────────────────────────────────────────────────────────────────────────┐
│  MOUSE INTERACTIONS:                                                    │
│  • Single Click on Pet      → Pet/Caress (be gentle!)                  │
│  • Double-Click on Pet      → Open chat interface                      │
│  • Click + Drag Pet         → Move pet around the screen               │
│  • Click on System Tray     → Show/Hide pet                            │
│  • Right-click System Tray  → Full options menu                        │
│                                                                         │
│  FEEDING YOUR PET:                                                      │
│  • Right-click tray icon → Food → Add Random Food                      │
│  • Watch your pet run to eat it automatically!                         │
│                                                                         │
│  MANUAL CONTROL:                                                        │
│  • Right-click tray icon → Remote Control                              │
│  • Use arrow keys or WASD to move your pet                             │
└─────────────────────────────────────────────────────────────────────────┘

TROUBLESHOOTING:
┌─────────────────────────────────────────────────────────────────────────┐
│  🔧 Pet doesn't appear:                                                │
│     • Check system tray for Minou icon                                 │
│     • Click the icon to show the pet                                   │
│     • Try restarting Minou                                             │
│                                                                         │
│  🔧 Chat doesn't work:                                                 │
│     • Make sure you're double-clicking the pet                         │
│     • The chat window may appear briefly then disappear - this is      │
│       normal, double-click again to reopen                             │
│                                                                         │
│  🔧 No sound:                                                          │
│     • Check Windows volume settings                                    │
│     • Right-click tray → Enable Audio                                  │
│                                                                         │
│  🔧 Performance issues:                                                │
│     • Right-click tray → Settings → Reduce animation speed             │
│     • Close other resource-intensive programs                          │
└─────────────────────────────────────────────────────────────────────────┘

UNINSTALLING:
If you ever need to uninstall Minou:
• Windows Settings → Apps → Minou → Uninstall
• Or Control Panel → Programs and Features → Minou
